# Программная документация: Система детекции клубники (ПП ОСК v1.0)

Данный документ описывает процесс установки, настройки окружения, а также инструкции по работе с моделями машинного зрения и графическим интерфейсом приложения. Система предназначена для потокового анализа изображений (видео, фото, веб-камера, RTSP), поиска ягод клубники, определения их спелости, ключевых точек (центр, шляпка, точка среза) и передачи координат роботу-манипулятору по протоколу UDP.

---

## 1. Системные требования

* **Операционная система:** Windows 10/11, Linux (Ubuntu 20.04+) или macOS.
* **Версия Python:** 3.8 — 3.11.
* **Оборудование:** * Для запуска в реальном времени рекомендуется наличие видеокарты (NVIDIA) с поддержкой CUDA.
    * Веб-камера (для тестирования захвата видео с устройства).

---

## 2. Установка и настройка окружения

### Шаг 1: Подготовка структуры проекта
Убедитесь, что файлы проекта расположены в следующей иерархии папок:
```text
project_root/
├── core/
│   ├── inference.py
│   └── video_worker.py
├── gui/
│   └── main_window.py
├── network/
│   └── transmitter.py
├── models/             <- Папка для весов нейросети
├── main.py
└── dummy_robot.py
```

### Шаг 2: Создание виртуального окружения
Откройте терминал (или командную строку) в корневой папке проекта (`project_root/`) и создайте изолированное виртуальное окружение:

**Для Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Для Linux / macOS:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### Шаг 3: Установка зависимостей
Установите необходимые библиотеки с помощью пакетного менеджера `pip`. В проекте используются фреймворк для интерфейса (PyQt6), библиотека компьютерного зрения (OpenCV), математический пакет (NumPy) и фреймворк нейросетей (Ultralytics YOLO).

Выполните команду:
```bash
pip install PyQt6 opencv-python numpy ultralytics
```
*(Опционально)* Если у вас есть видеокарта NVIDIA и вы хотите использовать ускорение CUDA, установите PyTorch с поддержкой CUDA согласно официальной инструкции на сайте pytorch.org перед установкой `ultralytics`.

## 3. Шаги работы с моделью (Настройка YOLO)

Система использует нейросеть архитектуры YOLO Pose для одновременного поиска объектов и их ключевых точек (Keypoint Detection).

1.  **Размещение весов:** Поместите обученный файл весов модели в папку `models/`. 
2.  **Именование файла:** По умолчанию в файле `main_window.py` ожидается файл `models/best.pt`. Если ваш файл называется иначе (например, `yolo_pose_best.pt`), переименуйте его или измените путь в файле `main_window.py` (строка 18):
    ```python
    self.detector = StrawberryDetector(model_path="models/ваше_название.pt")
    ```
3.  **Спецификация классов и точек:** Модель настроена на 2 класса: `0: "ripe"` (спелая) и `1: "unripe"` (неспелая). Для каждого объекта модель должна предсказывать 3 ключевые точки в строгом порядке:
    * `0`: Центр ягоды
    * `1`: Шляпка (Calyx)
    * `2`: Точка среза (Cut point)

## 4. Шаги работы с данными и интерфейсом (Запуск системы)

Система состоит из двух независимых компонентов: графического интерфейса системы технического зрения (ПК) и контроллера робота (манипулятора). Для локального тестирования предусмотрен эмулятор робота `dummy_robot.py`.

### Шаг 1: Запуск эмулятора робота
Откройте новое окно терминала (с активированным виртуальным окружением), перейдите в папку проекта и запустите эмулятор:
```bash
python dummy_robot.py
```
*Консоль выведет сообщение, что робот слушает локальный порт 5005 и ожидает координаты.*

### Шаг 2: Запуск главного приложения (GUI)
Откройте еще одно окно терминала, активируйте окружение и запустите главный файл приложения:
```bash
python main.py
```

### Шаг 3: Работа с интерфейсом
После запуска появится окно «ПП ОСК v1.0 — Система детекции клубники».

1.  **Выбор источника данных (Группа "Источник данных"):**
    * **Открыть фото:** Загрузка статического изображения (`.jpg`, `.png`). Обработка произойдет мгновенно без создания фонового потока.
    * **Открыть видео:** Выбор локального видеофайла (`.mp4`, `.avi`).
    * **Веб-камера:** Захват видеопотока с камеры устройства (индекс `0`).
    * **RTSP:** Введите сетевую ссылку на IP-камеру в поле ввода (например, `rtsp://admin:admin@192.168.1.100`) и нажмите «Подключить RTSP».
2.  **Настройка детекции (Группа "Параметры"):**
    * Используйте ползунок **Confidence**, чтобы менять порог уверенности нейросети в реальном времени. Если система выдает много ложных срабатываний, увеличьте значение (например, до `0.70`). Если пропускает ягоды — уменьшите (например, до `0.30`).
3.  **Анализ данных и логика отправки:**
    * В левой части экрана будет транслироваться размеченное видео. Зелеными рамками выделяются спелые ягоды, красными — неспелые. Точки среза, шляпки и центра соединены линиями.
    * В блоке **Статистика** отображается количество найденных ягод по категориям и задержка обработки кадра (latency ms).
    * **Логика робота:** Система отправит координаты манипулятору **только в том случае**, если ягода распознана как спелая (`ripe`) И нейросеть уверена в координатах точки среза более чем на 60% (`kpt_cut["conf"] > 0.6`).
4.  **Мониторинг логов:**
    * В текстовое поле под видео транслируются логи отправки данных. В окне терминала, где запущен `dummy_robot.py`, вы увидите подтверждение принятия команд `✅ ПРИНЯТА КОМАНДА: Движение манипулятора к X: ..., Y: ...`.
5.  **Остановка:** Для прекращения чтения видеопотока нажмите большую кнопку **ОСТАНОВИТЬ** в правом нижнем углу.
