# strawberry-picker-cv: ML-модель для роботизированного сбора клубники

Репозиторий содержит программное приложение для определения спелости клубники с помощью машинного обучения при роботизированном сборе. 

## 🍓 О проекте
Проект разрабатывается в рамках выпускной квалификационной работы. Главной целью является анализ изображений ягод клубники и выявление признаков спелости (по цвету, форме, текстуре и др.), а также нахождение точного места среза.
